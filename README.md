## 前后端分离博客-LBLOG

使用`vue` +` java` 开发学习。

#### 技术栈: 

1. 前端: `vue3` + `axios` +`vuex` +` vueRouter` + `elementui`
2. 后端:`java` + `shiro` +`jwt `
3. 数据库: `mysql`

#### 安装:

1.前端

```bash
npm install 
npm run serve
```

2.后端:

更新maven依赖，运行springboot

3.数据库：导入`lblog.sql`
